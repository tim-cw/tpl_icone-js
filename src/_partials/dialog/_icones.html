Icônes

  • Exportez les 3 images phone, mail et mailbox dans le dossier assets/icons
  • Assurez-vous d'avoir arrêté la tâche npm start (CTRL+C dans le terminal) et de la repartir ensuite pour bien créer le sprite svg avec vos nouvelles icônes.
  • Assurez-vous de respecter les bonnes pratiques de nomenclature des fichiers

Configuration des Icônes

  • Importer le fichier Icons dans votre JavaScript (il est dans un dossier utils )
  • Dans le fichier JavaScript, faites la commande pour charger le sprite icon (Icons.qqchose)
  • Chargez les styles des icônes dans le fichier main.scss (possiblement déjà une composante?)

Partials

  • Créez le dossier num1 dans le dossier _partials
  • Créez le fichier _ex1.html dans le dossier num1
  • À l'endroit désigné dans le HTML, faites l'inclusion de votre partial _ex1.html (voir notes du cours 2)
  • Faire votre HTML dans le nouveau fichier créé

HTML

  • Utilisez la structure .card>.card__media+.card__content
  • Vous devez utiliser le snippet tim-icon , pour mettre les icônes désirées
  • Le fond derrière l'icône n'est pas dans l'icône en tant que tel. Vous devrez ajouter un conteneur à l'icône

CSS

  • 3 colonnes égales de 1fr
  • Pour les cartes :
    • Une bordure: 1px, couleur tomato et solid
    • Marge interne: 30px (haut, droit, bas et gauche)
  • .card__media
    • Goutière de 20px entre l'icône et le texte
    • Largeur et hauteur de 50px pour l'icône, fond de couleur tomato
  • .card__content
    • Couleur des liens : tomato

Icônes

  • Exporter les 2 images success et error dans le dossier assets/icons
  • Les images sont fait avec des contours. Assurez-vous de les convertir avec la commande outline stroke dans figma
  • Assurez-vous de respecter les bonnes pratiques de nomenclature des fichiers

Configuration des Icônes

  • Assurez-vous d'avoir bien complété cette étape au précédent numéro

Partials

  • Créez le fichier _ex2.html dans le dossier num1
  • À l'endroit désigné dans le HTML, faites l'inclusion de votre fichier HTML
  • Faites votre HTML dans le nouveau fichier créé

HTML

  • Utilisez la même structure qu'au numéro précédent
  • Vous devez utiliser le snippet tim-icon , pour mettre les icônes désirées
  • Ajoutez le modificateur icon--lg , pour que l'icône soit plus grosse
  • Le fond derrière l'icône n'est pas dans l'icône en tant que tel. Vous devrez ajouter un conteneur à l'icône

CSS

  • 2 colonnes égales de 1fr
  • Pour les cartes :
    • Une bordure: 1px, couleur tomato et solid
    • Marge interne: 30px (haut, droit, bas et gauche)
    • Les éléments doivent être centré
  • .card__media
    • Goutière de 20px entre l'icône et le texte
    • Largeur et hauteur de 150px pour l'icone, fond de couleur tomato

Pour les aventuriers

Pour ceux qui veulent valider leur code et qui ont respecté les consignes de l'exercice précédent, vous pouvez faire ce numéro avec seulement quelques règles CSS et modificateurs.

  • Sur votre conteneur de cartes (.cards) vous pouvez ajouter un .cards--large et seulement y modifier le grid-template-column
  • Vous pouvez ensuite seulement changer le css pour votre icône et l'alignement du contenu lorsque la classe .cards--large est présente. (ex.: .cards--large .card__media)
