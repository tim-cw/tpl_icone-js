Étapes à suivre

Ajout d'un clic sur le bouton

  1. Reprendre le fichier Main complété au précédent numéro
  2. Dans la méthode init , créez une variable button
  3. Assignez lui la valeur du contenu HTML .js-button (querySelector ou querySelectorAll?)
  4. Sur votre bouton, ajoutez un événement de type click qui apelle la méthode toggleActive (n'oublier pas le this)
  5. Créez une méthode toggleActive
  6. Dans votre méthode toggleActive, ajoutez un console.log('je click sur mon bouton') et testez dans votre console de navigateur

On fait du deboggage

Avant d'aller plus loin, c'est important de tester au fur et à mesure. Si votre console.log dans la méthode toggleActive fonctionne, passez à l'étape suivante.. sinon trouvez l'erreur.

  1. La première étape est de regarder s'il y a un message d'erreur dans la console
  2. Ensuite, vous pouvez faire un console.log des éléments que vous avez créés comme la variable button
  3. La variable button devrait retourner la balise HTML button que vous venez de créer.
  4. Si ce n'est pas le cas, l'erreur est avec votre querySelector. Ajustez votre code au besoin.

On change la classe du texte

  1. Dans votre méthode toggleActive, créez une variable nommée text
  2. Assignez lui la valeur du contenu HTML de l'élément .js-text (querySelector ou querySelectorAll?)
  3. À l'aide de classList , faites alterner la classe active sur la variable text

Étapes à suivre

Ajout d'un click sur plusieurs boutons

  1. Reprendre le fichier Main complété du précédent numéro
  2. Ajoutez des classes préfixées (js-) pour les boutons et le H3 (.js-prof et .js-choice seront utilisés dans cette explication)
  3. Dans la méthode init , créez une variable buttons
  4. Assignez lui la valeur des éléments HTML qui ont la classe .js-prof (querySelector ou querySelectorAll?)
  5. À l'aide d'une boucle for , ajoutez un événement de type click qui apelle la méthode toggleContent sur tous vos boutons. (n'oublier pas le this)
  6. Créez la méthode toggleContent
  7. Dans votre méthode toggleContent , ajoutez un console.log('je click sur un bouton') et testez dans votre console de navigateur

On fait du deboggage

Avant d'aller plus loin, c'est important de tester au fur et à mesure. Si votre console.log dans la méthode toggleContent fonctionne, passez à la section suivante.

  1. La première étape est de regarder s'il y a un message d'erreur dans la console
  2. Ensuite, vous pouvez faire un console.log des éléments que vous avez créées comme la variable buttons
  3. buttons devrait retourner un tableau([] ou Array) de balises HTML (doit contenir 3 boutons)
  4. Si ce n'est pas le cas, l'erreur est avec votre querySelector

On change le contenu du texte

  1. Dans votre méthode toggleContent , créez la variable choice
  2. Assignez lui la valeur du contenu HTML .js-choice (querySelector ou querySelectorAll?)
  3. Avant d'aller plus loin, faites un console.log(texte du bouton cliqué). Voir innerText et currentTarget
  4. Changez le innerText de votre variable choice pour le innerText de l'élément cliqué
  5. À l'aide d'une concaténation, ajoutez devant la valeur du bouton cliqué : "Mon prof préféré est : "
